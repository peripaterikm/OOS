# Expected Semantic Clusters v1

Dataset v1 is intended to exercise 5-8 semantic clusters without implementing clustering in the fixture itself.

1. Workflow exception handling: `eval_v1_sig_001` through `eval_v1_sig_005`.
2. Compliance evidence collection: `eval_v1_sig_006` through `eval_v1_sig_008`.
3. Financial trust reconciliation: `eval_v1_sig_009`, `eval_v1_sig_010`, and `eval_v1_sig_022`.
4. Approval traceability: `eval_v1_sig_011`, `eval_v1_sig_012`.
5. Intake exception follow-up: `eval_v1_sig_013`, `eval_v1_sig_014`.
6. Renewal risk synthesis: `eval_v1_sig_016`, `eval_v1_sig_017`.
7. Physical ops exception audit: `eval_v1_sig_019`, `eval_v1_sig_020`.
8. Ambiguous/noisy parking candidates: `eval_v1_sig_015`, `eval_v1_sig_018`, `eval_v1_sig_021`.
