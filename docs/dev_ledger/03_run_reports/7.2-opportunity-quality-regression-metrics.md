## Limitations

1. Metrics are computed from synthetic evaluation dataset v1 (10 cases); rates reflect baseline behavior against labeled expectations, not production accuracy.
2. Unsupported assumptions are counted from opportunity candidate `unsupported_assumptions` fields only; dataset-level unsupported assumptions may be broader than what the model captures.
3. Duplicate cases are identified by `quality_label == 'duplicate_signal'`; near-duplicate evidence within a single case is tracked in risk_notes but not counted as separate duplicate cases.
4. Gate decisions are deterministic and may disagree with expected labels where the expected label reflects a desired outcome rather than the current gate logic output.
