## Purpose

Customer Voice Query generation expands discovery beyond founder/expert-language queries. It creates realistic user-language queries per topic and persona so future query planning can search for how operators actually describe pain, workarounds, buying intent, comparisons, and jobs-to-be-done.

## Product Correction

The implementation is role/persona-based, not owner-only. `ai_cfo_smb` includes owner/founder language, but also bookkeeper, accountant, fractional CFO, finance manager, operations manager, and freelancer/solo operator language.
