# Run Report 6.2 - Next Best Founder Actions

## Branch

`feat/v2-5-block-5-founder-feedback-loop`

## Summary

Implemented a deterministic next-best-founder-actions layer that converts the `WeeklyOpportunityReviewPackage` into a prioritized, flattened list of advisory `FounderAction` items. Each action includes a stable action ID, action type, title, rationale, priority, linked source artifact IDs, suggested next step, and an explicit `advisory_only=True` flag. The layer consumes the weekly review package dict (as produced by `weekly_review_package_to_json`) and produces a deterministic sorted list — no autonomous decisions, no portfolio transitions, no LLM/API calls.
