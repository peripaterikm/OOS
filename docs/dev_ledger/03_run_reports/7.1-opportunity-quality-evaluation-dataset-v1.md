## Implementation Details

### Dataset Location Decision

The canonical dataset lives at `examples/evaluation_dataset_v2_5/opportunity_quality_cases_v1.json`. This follows the existing convention:
- `examples/evaluation_dataset_v0/` — signals, expected clusters, expected opportunities, founder quality notes
- `examples/evaluation_dataset_v1/` — signals, expected clusters, expected opportunities, founder quality notes, expected idea quality notes
- `examples/evaluation_dataset_v2_5/` — opportunity quality cases v1 (new)

The loader default at [`EVALUATION_QUALITY_DATASET_V1_PATH`](src/oos/evaluation_dataset.py:13) resolves relative to the project root and points to this examples location. Tests import the constant rather than hardcoding a path.
