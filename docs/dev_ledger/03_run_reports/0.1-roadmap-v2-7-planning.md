## Key Planning Decisions

- **Traceability hardening first.** The source URL chain from `CandidateSignal.source_url` → … → `FounderFeedbackMapping.source_urls` must be complete; placeholder URNs are treated as bugs.
- **Additive, not breaking.** `linked_source_urls` defaults to `[]` on `FounderInboxReviewItem`; existing tests continue to pass.
- **Conditional scope for re-import policy.** Phase A (policy review) is always performed. Phase B (`--replace` implementation) only proceeds if justified.
- **Parallelizable.** Items 3.1 (policy review) and 4.1 (helper scripts) can run in parallel with items 1.1–2.1 (traceability).
- **Deterministic-first preserved.** All hardening maintains deterministic output; no live LLM/API calls by default.
- **Advisory-only preserved.** No autonomous portfolio transitions.
- **No new product layers.** v2.7 is hardening, not feature expansion.
- **Realistic 4–6 week scope.** 8 items, each with focused scope and clear acceptance criteria.
