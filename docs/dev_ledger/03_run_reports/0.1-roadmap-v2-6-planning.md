## Key Planning Decisions

- **Operationalize, don't rewrite.** Every v2.5 component has a module, model, and tests. v2.6 connects them.
- **Deterministic pipeline first.** The weekly cycle must complete without live LLM/API calls.
- **Markdown-first, JSON-backed.** Every founder-facing output has both formats.
- **Traceability preserved.** Input signal ID → inbox review item ID chain must be verifiable in tests.
- **Empty-state handling is first-class.** Every artifact and section must represent emptiness explicitly.
- **Advisory-only.** No autonomous portfolio transitions; 0 in validation.
- **Fixture-validated.** Item 8.1 provides the end-to-end regression gate.
- **Realistic 6–8 week scope.** 9 items, each with focused scope and clear acceptance criteria.
- **Parallelizable.** Items 3.1, 4.1, 6.1, and 7.1 can be built in parallel after 2.1.
