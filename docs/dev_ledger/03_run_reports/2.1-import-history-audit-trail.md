## Import History Schema

```python
@dataclass
class CorrectionEntry:
    correction_id: str            # SHA-256 composite hash[:12]
    corrected_at: str             # ISO 8601
    correction_mode: str          # "replace" | "amend"
    replaced_review_item_ids: list[str]
    old_decision_ids: list[str]
    new_decision_ids: list[str]
    old_artifact_checksums: dict[str, str]
    new_artifact_checksums: dict[str, str]
    warnings: list[str]
    errors: list[str]
    advisory_only: bool           # Always True
    no_live_api: bool             # Always True
    no_live_llm: bool             # Always True

@dataclass
class ImportHistoryLog:
    schema_version: str           # "import_history.v1"
    run_id: str
    entries: list[CorrectionEntry]
```

## Append-Only / Audit Behavior

- `_write_import_history()` loads existing history, appends entry, writes with `sort_keys=True`.
- Failed correction attempts never call `_write_import_history()` — no entries for failures.
- `read_import_history()` returns `None` for missing/malformed files.
- `build_import_history_summary()` returns empty defaults when history is absent.
