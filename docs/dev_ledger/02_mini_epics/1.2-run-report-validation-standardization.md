## Run-Report Standard

Canonical path:

```text
docs/dev_ledger/03_run_reports/<roadmap-item-slug>-validation.md
```

Required sections:

- Roadmap item
- Branch
- Commit / pre-commit state
- Commands run
- Results summary
- Blocked commands
- Known warnings
- No push / no merge / no tag / no release confirmation

Required command evidence includes focused tests when applicable, full unittest discovery, `.\scripts\oos-validate.ps1`, `.\scripts\verify.ps1`, and `git diff --check`.
