## Package JSON Behavior

Each weekly discovery run now writes:

```text
artifacts/discovery_runs/<run_id>/founder_discovery_package.json
```

The package includes run context, evidence and signal counts, source/classification/signal-type counts, top candidate signals, needs-human-review signals, recommended founder actions, limitations, and artifact paths.

## Package Markdown Behavior

Each weekly discovery run now writes:

```text
artifacts/discovery_runs/<run_id>/founder_discovery_package.md
```

The Markdown package includes:

- Executive summary
- Run context
- Signal overview
- Top candidate signals
- Needs human review
- Noise / low-signal summary
- Recommended founder actions
- Limitations

## Ranking Behavior

Candidate signals are ranked deterministically by:

1. confidence descending
2. signal-type priority: `buying_intent`, `pain_signal`, `workaround`, `competitor_weakness`, `trend_trigger`, `needs_human_review`
3. `signal_id` ascending

## Traceability Behavior

Top candidate signal package entries preserve:

- `signal_id`
- `evidence_id`
- `source_type`
- `source_url`
- `query_kind`

The package supports founder review but does not automate founder decisions.
