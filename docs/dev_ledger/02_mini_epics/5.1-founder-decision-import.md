## Goal

Allow the founder to record explicit decisions (PROMOTE/PARK/KILL/NEEDS_MORE_EVIDENCE/REVISIT_LATER) from the founder inbox v2 back into the system artifacts, updating the weekly run with validated decision records, feedback mappings, preference profiles, and parking lot records.

## Scope

- Core importer module: `src/oos/founder_decision_import.py`
- CLI subcommand: `import-founder-decisions-v2`
- Tests: `tests/test_founder_decision_import_v2.py` (41 tests)
- Docs: roadmap checklist, project state, mini-epic, run report

### Excluded

- No auto-generation or inference of decisions
- No portfolio state mutation
- No weekly_opportunity_review.json or next_best_actions.json rebuild (deferred to pipeline re-run)
- No live APIs, no LLM calls
- No status/dashboard integration

## Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Fail-closed** | If any input decision is invalid, no artifacts are written. Prevents half-imported founder state. |
| **Reject duplicates** | Duplicate `review_item_id` entries in the input are rejected. Safer than silent dedup or last-wins. |
| **Idempotent** | Re-importing decisions against already-decided opportunities fails cleanly. |
| **Advisory-only** | All artifacts carry `advisory_only: true`, `no_live_api: true`, `no_live_llm: true`. |
| **No auto-rebuild** | `weekly_opportunity_review.json` and `next_best_actions.json` are not rebuilt — the founder re-runs the weekly cycle builder. |

## Input Format

JSON array:
```json
[
  {
    "review_item_id": "inbox_review_...",
    "decision": "PARK",
    "reason_categories": ["unclear_buyer"],
    "notes": "Need 3 more buyer examples"
  }
]
```

JSONL (one object per line): same fields.

### Decision values

`PROMOTE` / `PARK` / `KILL` / `NEEDS_MORE_EVIDENCE` / `REVISIT_LATER`

Reason categories must match the existing `FounderDecisionV2` taxonomy per decision.

## CLI Usage

```powershell
python -m oos.cli import-founder-decisions-v2 `
  --project-root . `
  --run-id weekly_run_2026_01_01_abc123 `
  --decisions-file founder_decisions.json
```
