# 1.1 Evaluation Dataset v0

## Goal

Create an early repeatable evaluation fixture and reframe heuristic ideation as baseline/fallback/control.

## Context

Roadmap v2.2 needs measurable AI work before adding more AI-shaped stages.

## Inputs

- Active Roadmap v2.2 checklist
- Existing heuristic ideation path
- Repository fixtures and examples

## Outputs

- `examples/evaluation_dataset_v0/`
- Expected cluster, opportunity, and founder quality notes
- Heuristic generation mode labeling
- Focused dataset/baseline tests

## Key Decisions

- Synthetic signals are allowed only when explicitly labeled.
- Heuristic ideation is not treated as primary intelligence.

## Alternatives Rejected

- Waiting for a large real dataset before adding evaluation scaffolding.
- Making the heuristic generator smarter during this mini-epic.

## Validation

- Focused evaluation dataset tests.
- Full unittest discovery.
- `scripts/verify.ps1`.

## Known Limitations

- Dataset v0 is a smoke-test/calibration fixture, not a final benchmark.

## Next Step

Use the fixture to compare later AI stages and outputs.
