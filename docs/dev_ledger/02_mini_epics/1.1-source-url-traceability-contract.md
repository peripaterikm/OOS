## Goal

Define the canonical source URL traceability contract for the v2.6+ weekly loop. Create a validation scanner that detects placeholder URNs, missing source URLs, and malformed URLs across all pipeline artifacts without modifying any existing module.

## Scope

1. Audit every module in the weekly cycle pipeline that carries `source_urls` or `linked_source_urls`.
2. Define `SourceURLTraceabilityReport`, `SourceURLTraceabilityArtifactStatus`, and `SourceURLTraceabilityIssue` models.
3. Implement `check_source_url_traceability(run_dir)` to scan a weekly run directory.
4. Implement placeholder URN detection (`urn:oos:*`), missing URL detection, malformed URL detection.
5. Implement insufficient-evidence exemption for evidence packs.
6. Document the canonical traceability path from `CandidateSignal.source_url` through to `FounderFeedbackMapping.source_urls`.
7. Write focused tests (29 test cases, all pass).
8. Write contract document.
9. Keep everything deterministic, advisory-only, no live APIs/LLMs.

## Out of Scope

- No modification to any pipeline module (founder_inbox_v2, founder_decision_import, founder_feedback_mapping, weekly_cycle_builder).
- No `linked_source_urls` field added to `FounderInboxReviewItem` (that's 1.2).
- No placeholder URN removal from founder decision import (that's 1.3).
- No E2E traceability gate integration (that's 2.1).
- No artifacts written to `artifacts/`.
- No live APIs/LLMs.

## Key Models

- `SourceURLTraceabilityReport` — top-level report for one run
- `SourceURLTraceabilityArtifactStatus` — per-artifact status
- `SourceURLTraceabilityIssue` — single issue (placeholder/missing/malformed/unsupported)

## Key Functions

- `check_source_url_traceability(run_dir)` — scan a run directory
- `source_url_traceability_to_json(report)` — serialize report
- `is_placeholder_source_url(value)` — detect `urn:oos:*`
- `is_real_source_url(value)` — detect `http://` / `https://`
- `is_malformed_source_url(value)` — detect broken URLs
- `collect_source_urls_from_artifact(data, artifact_key)` — extract URLs from artifact dict
