Apply a narrow follow-up hardening patch after manual live smoke runs for Roadmap v2.4 item 1.1.

## Behavior

- Founder package top signals are deduplicated after ranking; `candidate_signals.json` still preserves all extracted signals.
- Dedup keeps the highest-confidence signal for each `source_type + source_url`, with `signal_id` as stable tie-breaker.
- Cleanup repairs common `вЂ...` mojibake fragments and removes broken emoji fragments from summaries.
- GitHub install guides, tutorial text, SEO/consulting copy, and generic marketing copy become `noise` or low-confidence review unless genuine user-pain markers are present.
- User-pain markers such as `Describe the problem`, `we would need to maintain a separate spreadsheet`, `I would like to`, and `manual spreadsheet` protect real issue reports from over-filtering.
